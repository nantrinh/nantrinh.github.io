Venice Framework

Venice is an open-source framework for deploying and managing stream processing pipelines.

Venice abstracts away much of the complexity of creating and managing a pipeline, allowing developers to focus on application code rather than infrastructure.

Venice's default pipeline consists of Apache Kafka brokers, Zookeeper, Zookeeper, Confluent Schema Registry, Kafka Connect, ksqlDB, PostgreSQL server, and Kafdrop.
